# 実験結果報告書

## 実験情報
- 実験タイトル: 新素材の耐久性試験
- 実験番号: TEST-2024-089
- 実験期間: 2024年1月15日〜1月24日
- 実験責任者: 材料研究部 佐藤健一
- 実験補助: 鈴木美咲、田中太郎

## 実験目的
新規開発材料「CompoundX-2024」の機械的強度および耐久性の評価

## 試験方法
### 引張試験
- 規格: JIS K 7161
- 試験片: ダンベル型5号
- 試験速度: 50mm/min
- 試験温度: 23°C、湿度50%

### 疲労試験
- 試験方法: 繰り返し曲げ試験
- 負荷: 最大応力の70%
- サイクル数: 100,000回

## 実験結果
### 引張強度
- CompoundX-2024: 95 MPa（n=10, σ=3.2）
- 従来材料: 78 MPa（n=10, σ=2.8）
- 改善率: +22%

### 疲労寿命
- CompoundX-2024: 100,000サイクル後も破断なし
- 従来材料: 平均85,000サイクルで破断

## 結論
CompoundX-2024は従来材料と比較して優れた機械的特性を示した。
実用化に向けて次フェーズの試験に進む。
