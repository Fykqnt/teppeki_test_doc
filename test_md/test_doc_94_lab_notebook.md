# 実験ノート

## 実験者情報
- 氏名: 鈴木美咲
- 所属: バイオテクノロジー研究所 分子生物学部門
- 実験日: 2024年1月24日

## 実験タイトル
遺伝子組換えタンパク質の発現確認実験（実験No. EXP-2024-056）

## 実験目的
pET-28a(+)ベクターに挿入した目的遺伝子の大腸菌での発現確認

## 実験材料
- 菌株: E.coli BL21(DE3)
- プラスミド: pET-28a-target gene
- 培地: LB培地（カナマイシン 50μg/mL）
- 誘導剤: IPTG 1mM

## 実験手順
1. 前培養（37°C、16時間）
2. 本培養（37°C、OD600=0.6まで）
3. IPTG添加（最終濃度1mM）
4. 培養継続（37°C、4時間）
5. 菌体回収、SDS-PAGE解析

## 実験結果
SDS-PAGEにて約45kDaの位置にバンドを確認
予想分子量（48kDa）と概ね一致
発現量: 総タンパク質の約30%

## 考察
目的タンパク質の発現を確認。次回は精製を実施予定。
